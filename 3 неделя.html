Отчет по учебной практике

Данное приложение ориентированно для людей совершающих авиаперелеты по всему миру. Пассажирам, которые нуждаются в ознакомлении со статусом своего рейса прямо из своего мобильного телефона.

Задача мобильного приложения в том, чтобы вносить пассажиров в базу данных SQL по их имени, фамилии и номеру рейса. После внесения информации человек получает всю информацию о своём рейсе и состоянии регистрации на него.

Ниже вы можете ознакомиться с внешним видом приложения, для вас представлены окна приветствия и регистрации.
