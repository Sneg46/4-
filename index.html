Отчет по учебной практике

Учебная практика по дисциплине разработка модулей программного обеспечения для компьютерных систем, состоит из 4 модулей, каждый из которых содержит индивидуальные задания, которые по окончании практики должны быть представлены в виде отчёта (сайта, разработанного в заключительном модуле практики).

В ходе учебной практики были решены след вопросы: решение задач, разработка АИС, разработка мобильного приложения, создание сайта и оценка эффективности.

- Первый модуль это решение различных задач, на выбранном нами языке программирования.

- Второй модуль состоит из разработки автоматизированной информационной системы аэропорт.

- Третий модуль это создание мобильного приложения.

- Четвертый этап, создание сайта.
