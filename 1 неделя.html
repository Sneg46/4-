Решение задач

1. Уплотнить матрицу, удаляя из нее строки и столбцы, заполненные нулями.

2. Определить класс Квадратное уравнение. Реализовать методы для поиска корней. Создать массив/список/множество объектов и определить наибольшие и наименьшие по значению корни.

3. Создать объект класса Дом, используя классы Окно, Дверь. Методы: закрыть на ключ, вывести на консоль количество окон, дверей.

4. Создать класс Computer с внутренним классом, с помощью объектов которого можно хранить информацию об операционной системе, процессоре и оперативной памяти.

5. Напечатать квитанцию об оплате телеграммы, если стоимость одного слова задана.

6. Создать объект класса Дом, используя классы Окно, Дверь. Методы: закрыть на ключ, вывести на консоль количество окон, дверей. Реализуя собственные обработчики исключений, исключения ввода/вывода.
